Unityコース教科書（2D横スクロールアクション）

今回のUnityコースはこちらを使用します

教材には元教材の実スクリーンショット・操作動画を掲載しています。使用バージョンは Unity 2022.3 LTS（Platformer MicrogameはUnity 6では動作しないため）。講習日程は決まりしだい各章に記載します。

この教科書は、ゲームエンジン Unity を使って Platformer Microgame（2Dアクションゲーム）の改造を1本完成させるための教材です。 毎回「動くもの」が増えていき、最終回には自分だけのゲームができあがります。 講習で使うだけでなく、家でも自分のペースで読み進められるように作られています。

🎯 Unity編のゴール

  • Unityで2Dアクションゲームを1本完成させる
  • C#スクリプトで「自分の思いどおりの動き」を作れるようになる
  • エラーが出ても、自分で調べて直せるようになる

目次

講習前に家で 第0章 事前準備 〜Unityをインストールしよう〜
Unity Hub と Unity 2022.3 LTS を家でインストール。アカウント作成から動作確認まで
 第1回 第1章 ゲームを手に入れて遊ぼう 〜Platformer Microgameをはじめる〜
Platformer Microgame を手に入れて遊ぼう。エディタ画面の見方も覚える
 第2回 第2章 ビルボードとゲーム名 〜自分の画像とタイトルでオリジナルゲームにしよう〜
ビルボードに自分の画像を飾り、ゲーム名とフォントでオリジナル化
 第3回 第3章 世界の色を変えよう 〜色で自分だけの世界観をつくる〜
タイルの色・トレイル・空の色で、自分だけの世界観をつくる
 第4回 第4章 世界に命を吹き込もう 〜揺れる木と紙吹雪のしかけ〜
揺れる木と、通ると紙吹雪が舞う「見えないしかけ」を作る
 第5回 第5章 ステージを作りかえよう 〜タイルパレットとパワーアップ〜
タイルパレットでステージを描き変え、Speed Pad / Bounce Pad を追加
 第6回 第6章 仕上げて発表しよう 〜ワールドを飾って、自分だけの作品に〜
装飾ワザで仕上げて自由カスタマイズ。最後はみんなで発表
