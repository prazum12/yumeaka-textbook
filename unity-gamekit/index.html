Unityコース教科書（2D Game Kit版）

これは自由研究用の教材です

今回の講習で使うのはPlatformer Microgame版です。この2D Game Kit版は講習では扱いませんが、「自分でアセットを追加してもっと作り込みたい」という人向けに公開しています。興味がある人は自分のペースで進めてみよう。

この教科書は、Unity公式の「2D Game Kit」を使って、 主人公エレン（Ellen）が冒険する2Dアクションゲームの自分だけのステージを作る教材です。 プログラミングは使わず、置いて・つないで・調整するだけで、動くしかけがどんどん増えていきます。 講習で使うだけでなく、家でも自分のペースで読み進められます。

🎯 このコースのゴール

  • Unityエディタの基本操作を身につける
  • 動く床・扉・敵・テレポートなどのしかけを自分で組み込める
  • しかけを組み合わせた自分だけのステージを完成させて発表する

目次

講習前に家で 第0章 事前準備 〜Unityをインストールしよう〜
Unity Hub と Unity 2022.3 LTS を家でインストール。アカウント作成から動作確認まで
 第1回 第1章 ゲームキットを手に入れて遊ぼう 〜2D Game Kitをはじめる〜
2D Game Kitをインポートして、サンプルゲームThe Explorerで遊ぶ。エディタ画面の見方
 第2回 第2章 自分のステージを作ろう 〜タイルでペイント〜
Kit Toolsで新規シーンを作り、タイルパレットでステージを描いてテストプレイ
 第3回 第3章 動くしかけを作ろう 〜動く床とイベントで開く扉〜
MovingPlatformの経路づくりと、感圧パッドで扉が開くイベントのつなぎ方
 第4回 第4章 敵とダメージのしくみ 〜ChomperとSpitterを攻略せよ〜
敵の視野や速さを調整し、箱を落として倒すダメージのしくみを作る
 第5回 第5章 世界を飾ってワープさせよう 〜装飾とテレポート〜
装飾スプライトで見た目を仕上げ、テレポートで隠し部屋や別エリアへ
 第6回 第6章 仕上げて発表しよう 〜自分だけのゲームを完成させる〜
Zone1〜5からアイデアを集め、しかけを組み合わせて完成→発表

この教科書の使い方

マーク意味
🎯 ゴールこの章でできるようになること。最初に読もう
🔢 手順番号の順番どおりに操作すればOK
💪 やってみよう自分の力で挑戦する課題。ヒントは折りたたみの中
🚨 つまずきポイントうまくいかないときは、まずここを見る
🤖 AIの使い所ChatGPTなどのAIを学習に活かすコツ。丸写しではなく「先生」として使おう
✅ チェックリスト章の最後に、できたか自分で確認
